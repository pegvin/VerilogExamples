# VerilogExamples
Examples of various combinational circuits in verilog

**Note**:
- The code is split in two files, `.tb.v` is the
  testbench & Just regular `.v` is the DUT.
- To run the code in tools like EDA playground, You will
  have to remove the lines starting with:
  ```
  `include ...
  ```
